# Terradiscover Frontend

## How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/JordanMarcelino/terradiscover-frontend.git

cd terradiscover-frontend
```

### 2. Copy and Modify Env Example

```bash
cp .env.example .env
```

Modify the `.env`

### 3. Build and Run

```bash
npm install
npm run dev
```
